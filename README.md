# koa-mysql-template

#### 项目介绍
koa2 + mysql + typescript项目框架


#### 目录结构
```
koa-mysql-template
├── README.md
├── docker           // 支持用docker启动mysql
├── nodemon.json     //  nodemon配置
├── package-lock.json
├── package.json
├── src
    ├── config       // 项目配置 
    ├── controller   
    ├── main.ts      // 主入口
    ├── middleware
    ├── model
    ├── router       // 路由
    ├── service
    └── test
├── text.txt
└── tsconfig.json
```
